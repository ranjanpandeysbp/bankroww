## Want to run locally?

I am using CRA for this repo, clone this code by using:
`git clone https://github.com/shuvamk/bankroww.git`
then go to directory:
`cd bankroww`
install npm packages:
`npm i`
run the server:
`npm start`
The project will be (in most caes)started on localhost:3000
